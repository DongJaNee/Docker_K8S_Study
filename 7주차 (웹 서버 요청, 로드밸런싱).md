## 쿠버네티스 기반 웹 서버 요청 테스트 및 로드 밸런싱 확인
```
# Pod의 IP 주소로 직접 웹서버 요청 테스트
$ kubectl describe pod web-server | grep IP | head -1
$ curl 10.42.0.48
```
